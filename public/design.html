形狀與陰影層級 Shapes & Elevation

全域規則 border-radius: 0px ({rounded.none})，所有元件皆為硬角矩形。深度以硬邊像素陰影（無模糊）呈現。

{elevation.pixel-sm}

適用於 Toast、Key chips 等輕量懸浮元件。

box-shadow: 2px 2px 0 0 var(--canvas-deep)

{elevation.pixel-md}

適用於按鈕、預設面板等中量級深度元件。

box-shadow: 4px 4px 0 0 var(--canvas-deep)

{elevation.pixel-lg}

適用於命令面板 Cmd+K、聚焦面板等重量級懸浮元件。

box-shadow: 8px 8px 0 0 var(--canvas-deep)

02 — Typography Scale

VT323 顯示字，IBM Plex Mono 操作字。

文字本身就是終端機的品牌資產。顯示層保持 pixel flavor，操作層保持單寬可讀性；不使用斜體、不使用 500 或 700。

08 — Responsive Rules

桌面優先，手機優雅降級。

daily-tools 是寬螢幕工作介面。小尺寸保持功能可讀，但不假裝每個工具都能在 320px 下完整展示。

版面斷點與行為
斷點 寬度 行為
Mobile ≤ 639px 單欄、breadcrumb 隱藏 category、dual-pane 改垂直堆疊。
Tablet 640–1023px 側欄改為上方索引區，主要畫布全寬。
Desktop ≥ 1024px 240px 固定側欄，prompt bar sticky，工具面板雙欄。
Wide ≥ 1440px 維持密集資訊配置，不額外增加 marketing whitespace。
