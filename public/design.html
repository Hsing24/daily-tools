01 — Color Palette

一個 mint action，三個語意訊號。

參考頁以藍色統一互動，本頁改以 Pantone 333c 等價的 mint 綠承擔所有互動狀態。Gold 只做狀態，Red 只做破壞或錯誤。

Canvas

#0A1A2F

主畫布。所有工具頁都從這個深藍終端底色開始。

Elevated

#122339

工具面板與重點容器。深度由表面階層提供，不靠陰影。

Nested

#15263F

面板內的鍵帽、列表選取列、次級資料槽。

Primary Mint #3FE0C5

所有連結、焦點、主操作、prompt glyph。

Status Gold #F1B434

非阻斷狀態、次要訊息、breadcrumb category。

Warning Red #E03C31

錯誤、重設、破壞性操作，不做品牌裝飾。

Canvas Deep #050E1C

側欄、prompt bar、footer、modal scrim 基底。

Canvas #0A1A2F

主內容背景，承接 CRT scanline overlay。

Elevated #122339

工具面板、Cmd+K modal、active nav item。

Input Slot #081424

文字輸入與 textarea 的凹槽色。

Ink Bright #E6F1FF

標題、強調文字、鍵帽文字。

02 — Typography Scale

VT323 顯示字，IBM Plex Mono 操作字。

文字本身就是終端機的品牌資產。顯示層保持 pixel flavor，操作層保持單寬可讀性；不使用斜體、不使用 500 或 700。

03 — Component Variants

按鈕是實心像素塊，面板是硬邊工具槽。

參考頁的 pills 與圓角卡片在這裡被替換成終端語法：borderless button、2px panel、4px focus/modal。

Buttons

操作色只有 mint。Gold 與 red 只能表達語意狀態。

Tool Panel

標準工具容器。背景固定 elevated，chunky border 是主要結構線。

06 — Command Palette

Cmd+K 是主要入口，不是附加捷徑。

每個工具兩次按鍵內可抵達。modal 使用 4px mint border，列表選取列使用 primary-dim 與左側 4px 狀態條。

08 — Responsive Rules

桌面優先，手機優雅降級。

daily-tools 是寬螢幕工作介面。小尺寸保持功能可讀，但不假裝每個工具都能在 320px 下完整展示。

版面斷點與行為
斷點 寬度 行為
Mobile ≤ 639px 單欄、breadcrumb 隱藏 category、dual-pane 改垂直堆疊。
Tablet 640–1023px 側欄改為上方索引區，主要畫布全寬。
Desktop ≥ 1024px 240px 固定側欄，prompt bar sticky，工具面板雙欄。
Wide ≥ 1440px 維持密集資訊配置，不額外增加 marketing whitespace。
